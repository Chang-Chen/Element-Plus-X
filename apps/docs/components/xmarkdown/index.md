# Welcome 欢迎 🌹

## 介绍

`Welcome` 这个组件可以清晰传达给用户可实现的意图范围和预期功能。使用合适的欢迎推荐组件，可以有效降低用户学习成本，让用户快速了解并顺利开始。

## 属性

| 属性名      | 类型   | 是否必填 | 默认值 | 描述          |
| ----------- | ------ | -------- | ------ | ------------- |
| `markdown`  | string | 是       | ''     | markdown 内容 |
| `allowHtml` | bool   | 是       | ''     | markdown 内容 |

## 插槽

| 插槽名   | 参数 | 类型 | 描述               |
| -------- | ---- | ---- | ------------------ |
| `#image` | -    | Slot | 自定义欢迎图片内容 |
